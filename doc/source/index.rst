Welcome to Sympy Plotting Backends's documentation!
===================================================

This Python module contains a few plotting backends that can be used with
`SymPy <http://github.com/sympy/sympy/>`_ . A backend represents the plotting
library: it provides the necessary functionalities to quickly and easily plot
the most common types of symbolic expressions, such as line plots, surface
plots, parametric plots, vector plots, complex plots, geometric plots.
